# Prompt engineering logg
Under utvecklingen av mitt Java-projekt använde jag ChatGPT som stödverktyg i flera steg. Jag formulerade en serie promptar (frågor och uppgifter) för att få hjälp med specifika delar av kodutvecklingen.

I första hand bad jag ChatGPT att visa exempel på hur vissa delar av koden kunde implementeras steg för steg. Dessa exempel använde jag som referens när jag själv skrev och anpassade min egen kod till projektet.

Jag använde även ChatGPT för att få en djupare förståelse av SOLID-principerna, särskilt Single Responsibility Principle. Jag bad om förklaringar på hur dessa principer fungerar och hur de kan tillämpas praktiskt i en Java-applikation. Utifrån svaren fick jag en tydligare bild av hur jag kunde strukturera mitt projekt på ett mer modulärt och hållbart sätt.

ChatGPT användes alltså som ett pedagogiskt stöd — för att generera exempel, förklara designmönster och principer, samt hjälpa till med idéer kring hur projektets arkitektur kunde förbättras — men den slutliga koden och strukturen har jag själv skrivit och anpassat till mitt eget projekt.
